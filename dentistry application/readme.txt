Проект представляет собой приложение для стоматологии для трех действующих лиц - стоматолога, пациента и работника регистратуры.


	Работник регистратуры выполняет следующие функции:
•	создание записи на первичный осмотр ротовой полости;
•	внесение и удаление данных о пациентах;
•	просмотр клиентской базы.

	Функциями стоматолога являются:
•	просмотр всех приемов стоматологической клиники;
•	запись (в том числе новых) пациентов на прием;
•	просмотр и редактирование своих услуг;
•	просмотр и редактирование своих приемов.

	Пациент выполняет следующие функции:
•	просмотр услуг, предоставляемых стоматологической клиникой;
•	просмотр списка сотрудников;
•	просмотр полученных услуг.


Все это реализовано в проекте с помощью средств Microsoft Visual Studio и Microsoft SQL Management Studio. Были использованы языки C#, SQL, T-SQL (хранимые процедуры и функции).
